# Fun Project: Number Guessing Game

## Description
This is a simple Python-based number guessing game. The program generates a random number between 1 and 100, and the user tries to guess it.

## Features
- Random number generation
- User input handling
- Feedback (Too high / Too low)
- Tracks number of attempts

## Tools Used
- Python

## Conclusion
This project demonstrates basic Python concepts like loops, conditionals, and user interaction.
