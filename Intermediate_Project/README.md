# Intermediate Project: Student Performance Analysis

This project analyzes student marks using Python. It performs basic data analysis and visualization to understand student performance.

- Source: Self-created dataset
- Data includes student names and their marks

1. Data creation
2. Data analysis using pandas
3. Visualization using matplotlib

## Results
- Compared student performance
- Visualized marks using a bar chart

## Tools Used
- Python
- Pandas
- Matplotlib

## Conclusion
This project shows how data analysis and visualization can help in understanding performance trends.
